# **项目描述**

基于nest框架、pnpm构建工具
## Pre-request（前置要求）

- NodeJs >= 14.19.0
- pnpm

## 开发环境安装

```bash
$ pnpm install
```

## 启动

```bash
# development
$ pnpm start

# watch mode
$ pnpm start:dev

# production mode
$ pnpm start:prod
```

## 跑单元测试

```bash
# unit tests
$ pnpm test

# e2e tests
$ pnpm test:e2e

# test coverage
$ pnpm test:cov
```
